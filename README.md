# tnagata.github.io
すべてのリポジトリにソースコードとその解説があるので、教育用、自己学習用に使えます

1. demo application
- llamabank-react : React, TypeScript, Go, Gin, Sqlite3を使ったラマ銀行のデモアプリ
- llamabank-rust : Dioxus, Axumを使ったラマ銀行のデモアプリ(純rustアプリ）
- llamabank-android : Kotlinを使ったラマ銀行のデモアプリ(バックエンドは上記のaxumそのまま）
  
2. Rust学習用
- command-line-rust (「Rustの練習帳」のサンプルコードが最新のRustでエラーやワーニングを出ない
- mandelbrot (マルチスレッドの学習用、Jim Blandy氏のサンプルコードをアップデート（最新Rust環境でのビルドエラー修正と処理時間測定
  コードの追加）
